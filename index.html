O abnt-latex \(\ll\)á-bé-ene-tê lah-tech\(\gg\) é um conjunto de recursos para facilitar a escrita de documentos escritos em \(\LaTeX\) seguindo as normas da ABNT (Associação Brasileira de Normas Técnicas).

O conjunto abnt-latex é composto por uma classe (abntlatex.cls), por pacotes de citação, de formatação de estilos bibliográficos (abnt.bst), de uma imagem Docker com TeX Live e R e um language server para \(\LaTeX\).

Visite a página do abnt-latex no Github.
